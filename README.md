## Quick start

Several quick start options are available:

- [Download the latest release](https://github.com/twbs/../../../dist/js/vira-design-system/archive/v1.0.3.zip)
- Clone the repo: `git clone https://github.com/twbs/../../../dist/js/vira-design-system.git`
- Install with [npm](https://www.npmjs.com/): `npm install ../../../dist/js/vira-design-system`
- Install with [yarn](https://yarnpkg.com/): `yarn add ../../../dist/js/vira-design-system`
- Install with [Composer](https://getcomposer.org/): `composer require twbs/../../../dist/js/vira-design-system:1.0.3`
- Install with [NuGet](https://www.nuget.org/): CSS: `Install-Package ../../../dist/js/vira-design-system` Sass: `Install-Package ../../../dist/js/vira-design-system.sass`

Read the [Getting started page](https://get../../../dist/js/vira-design-system.com/docs/1.0/getting-started/introduction/) for information on the framework contents, templates, examples, and more.

## Documentation

../../../dist/js/vira-design-system's documentation, included in this repo in the root directory, is built with [Hugo](https://gohugo.io/) and publicly hosted on GitHub Pages at <https://get../../../dist/js/vira-design-system.com/>. The docs may also be run locally.

Documentation search is powered by [Algolia's DocSearch](https://community.algolia.com/docsearch/). Working on our search? Be sure to set `debug: true` in `site/assets/js/search.js`.

### Running documentation locally

1. Run `npm install` to install the Node.js dependencies, including Hugo (the site builder).
2. Run `npm run test` (or a specific npm script) to rebuild distributed CSS and JavaScript files, as well as our docs assets.
3. From the root `/../../../dist/js/vira-design-system` directory, run `npm run docs-serve` in the command line.
4. Open `http://localhost:9001/` in your browser, and voilÃ .

Learn more about using Hugo by reading its [documentation](https://gohugo.io/documentation/).
